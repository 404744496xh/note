#### <font style="color:rgb(79, 79, 79);">主要特点</font>
+ **<font style="color:rgb(77, 77, 77);">多格式支持</font>**<font style="color:rgb(77, 77, 77);">：轻松转换 PDF、Word、PowerPoint、Excel、图片、音频、HTML 等多种格式</font>
+ **<font style="color:rgb(77, 77, 77);">结构保留</font>**<font style="color:rgb(77, 77, 77);">：自动保留标题、列表、表格、链接等重要文档结构</font>
+ **<font style="color:rgb(77, 77, 77);">MCP 协议</font>**<font style="color:rgb(77, 77, 77);">：提供 Model Context Protocol 服务器，可直接与 Claude Desktop 等 LLM 应用无缝集成</font>
+ **<font style="color:rgb(77, 77, 77);">AI 友好</font>**<font style="color:rgb(77, 77, 77);">：生成的 Markdown 格式对大语言模型特别友好，提高处理效率</font>
+ **<font style="color:rgb(77, 77, 77);">插件扩展</font>**<font style="color:rgb(77, 77, 77);">：支持第三方插件，可以扩展更多功能</font>
+ **<font style="color:rgb(77, 77, 77);">集成能力</font>**<font style="color:rgb(77, 77, 77);">：可与 Azure Document Intelligence 集成，提高复杂文档的处理能力</font>
+ **<font style="color:rgb(77, 77, 77);">图像描述</font>**<font style="color:rgb(77, 77, 77);">：支持使用大语言模型为图像生成描述，增强内容理解</font>
+ **<font style="color:rgb(77, 77, 77);">轻量灵活</font>**<font style="color:rgb(77, 77, 77);">：按需安装依赖，只选择需要的功能模块</font>

<font style="color:rgb(77, 77, 77);">Markdown 提供了一种简洁、易学、适用于技术文档和代码的文本标记方式。它有助于程序员更轻松地记录和分享他们的工作，提高了文档的可读性和可维护性，使得技术沟通变得更加高效。</font>
