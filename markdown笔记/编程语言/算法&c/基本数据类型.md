数据类型	占用空间	取值范围

short(短整型)	2字节	(-2<sup>15 ~ 2</sup>15-1)

int(整型)	4字节	(-2<sup>31 ~ 2</sup>31-1)

long(长整形)	Windows为4字节，Linux为4字节(32位)，8字节(64位)	(-2<sup>31 ~ 2</sup>31-1)

long long(长长整形)	8字节	(-2<sup>63 ~ 2</sup>63-1)

| 数据类型 | 占用空间 | 有效数字范围 |
| --- | --- | --- |
| float | 4字节 | 7位有效数字 |
| double | 8字节 | 15～16位有效数字 |
